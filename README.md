## ğŸ“ Mathematical Model

The asset price follows **Geometric Brownian Motion**:

[
S_{t+\Delta t} = S_t \cdot \exp\left((r-\tfrac{1}{2}\sigma^2)\Delta t + \sigma\sqrt{\Delta t}Z\right)
]

Where:

* (S_t) â†’ Current asset price
* (r) â†’ Risk-free interest rate
* (\sigma) â†’ Volatility
* (Z) â†’ Standard normal random variable

1. Simulate (N) stochastic price paths
